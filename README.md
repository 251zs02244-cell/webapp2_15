# Paint Inventory

絵の具の在庫を管理するためのシンプルなWebアプリです。  
絵の具名と色を登録し、一覧で確認・削除ができます。

## 機能

- 絵の具名と色を登録
- 色をサークル表示で確認
- 不要な絵の具の削除
- データをブラウザの localStorage に保存

## 使い方

1. 絵の具名を入力する
2. 色を選択する
3. 「追加」ボタンを押す
4. 一覧に表示される
5. 削除は ✖ ボタンを押す

登録したデータはブラウザに保存され、ページを再読み込みしても保持されます。
